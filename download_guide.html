Download Guide

Archived Videos are available on an S3 bucket that can be accessed via Globus. You will need a Globus account. You will be prompted to make a Globus account upon clicking a download link

Step 1

Click the download link from the project dropdown. Or you can use the example link at the top of this page

Step 2: Open Globus

The files to be downloaded will be shown on the left. Make sure that these files/folders are the correct type of file. Additionally, ensure that your computer has enough storage to download the desired files/folders.

Click the search bar to find your computer's local collection

Step 3: Find your local machine and select it as the target collection

In the list of collections, select your local machine. If you have not created any globus collections, it should be your only collection

Step 4: Select a directory on your machine

Choose where you want to save the files that you are downloading from Globus.

Step 5: Select the files that you want to transfer

Click the checkbox to the right of each file/folder you want to download

Step 6: Start the transfer

Click the large blue start button. Downloading folders this large will likely take a long time

Step 7: After waiting a little bit, ensure that the download is proceeding correctly

First select the Activity Tab on the right side, then check the transfer with the blue "in progress" icon as shown in the picture. If there is a yellow "warning" indicator, then the transfer is currently experiencing an error. If it is a red X, it has failed. If it is a green check mark, it has completed correctly

Step 8: Open the details for the specific transfer

Click the name of the transfer that you just started

Step 9: Find the Event Log page

Click event log on the details page for the transfer in question

Step 10: Analyze the event logs

In the "progress" event, you will be able to see the number of bytes transferred during a specific range of time In the "succeeded" event, you will be able to see the number of files transferred during a specific range of time "file not found" and "permission denied" are the most common failure messages. Double check that you used the right link and are signed in to a Globus account
