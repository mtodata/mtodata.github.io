Density Coordinated Ramp Control Field Implementation

May 2012 - June 2015 | by John Hourdos, Nikolas Geroliminis, Stephen Zitzow, and Ypatia Stefania Limniati

Purpose:

This project evaluated the UMD KAdaptive ramp metering algorithm on Twin Cities freeways, ultimately replacing the existing Stratified Zone algorithm system-wide, while plans to test the UMN Density algorithm were deferred due to technical and scheduling issues.
